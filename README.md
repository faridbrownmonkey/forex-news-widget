# 📰 Forex News Impact Widget

A lightweight single-file widget for quickly assessing how upcoming forex/XAUUSD news events are likely to impact price — built as a fast pre-trade check rather than a full dashboard.

## Problem

Before entering a trade, I wanted a quick way to check whether a major news event was imminent and what its likely directional impact would be, without opening a full economic calendar site.

## Features

- Compact news impact view focused on XAUUSD and major forex pairs
- Quick-reference layout designed for a fast pre-trade check, not deep research

## Tech Stack

- Vanilla HTML, CSS, JavaScript — single file, no dependencies

## Why I built it this way

This is intentionally minimal — a companion tool to the larger Forex Intel dashboard, built for the moments when I just need a fast yes/no check before clicking buy or sell.
